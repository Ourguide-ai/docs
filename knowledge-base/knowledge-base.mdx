Add content your agent can reference when answering questions.

## Sources

| Source      | Best for                                    | Updates                    |
| ----------- | ------------------------------------------- | -------------------------- |
| **Files**   | PDFs, manuals, docs                         | Manual re-upload           |
| **Website** | Public docs, help centers                   | Manual re-crawl            |
| **Notion**  | Internal wikis, frequently updated content  | Live (always current)      |

## How it Works

1. User asks a question
2. Ourguide searches your knowledge base for relevant content
3. Agent uses that content to formulate an accurate answer

<Info>
  This is called **RAG** (Retrieval-Augmented Generation).
</Info>

## Combining Sources

Use multiple sources together for comprehensive coverage:

- **Files** for product manuals
- **Website** for your public documentation
- **Notion** for internal procedures
