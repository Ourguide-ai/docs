The **Model Context Protocol (MCP)** allows you to connect AI coding agents to Ourguide's documentation. Once connected, your agent can help you integrate Ourguide into your codebase with full context of our APIs and features.

## Server URL

Add this URL to your coding agent's MCP configuration. Works with **Cursor**, **Claude**, **Windsurf**, and any MCP-compatible tool.

```
https://docs.ourguide.ai/mcp
```

<Tip>
  Using Cursor? You can install the MCP server in one click from the dashboard.
</Tip>

## What you can do

With Ourguide Docs MCP connected, your coding agent can:

- Help integrate the Ourguide widget into your website
- Set up identity verification for authenticated users
- Configure OpenAPI or MCP server integrations
- Understand how to use Ourguide's features

## Setup

### Cursor

Add the following to your `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "ourguide-docs": {
      "url": "https://docs.ourguide.ai/mcp"
    }
  }
}
```

### Claude Desktop

Add to your Claude Desktop MCP config:

```json
{
  "mcpServers": {
    "ourguide-docs": {
      "url": "https://docs.ourguide.ai/mcp"
    }
  }
}
```

### Other MCP-compatible tools

Use the server URL `https://docs.ourguide.ai/mcp` in any tool that supports the Model Context Protocol.
