The **Model Context Protocol (MCP)** allows you to connect AI coding agents to Argide documentation. Once connected, your agent can help you integrate Argide into your codebase with full context of our APIs and features.

## Server URL

Add this URL to your coding agent's MCP configuration. Works with **Cursor**, **Claude**, **Windsurf**, and any MCP-compatible tool.

```
https://docs.argide.ai/mcp
```

<Tip>
  Using Cursor? You can install the MCP server in one click from the dashboard.
</Tip>

## What you can do

With Argide Docs MCP connected, your coding agent can:

- Help integrate the Argide widget into your website
- Set up identity verification for authenticated users
- Configure OpenAPI or MCP server integrations
- Understand how to use Argide's features

## Setup

### Cursor

Add the following to your `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "argide-docs": {
      "url": "https://docs.argide.ai/mcp"
    }
  }
}
```

### Claude Desktop

Add to your Claude Desktop MCP config:

```json
{
  "mcpServers": {
    "argide-docs": {
      "url": "https://docs.argide.ai/mcp"
    }
  }
}
```

### Other MCP-compatible tools

Use the server URL `https://docs.argide.ai/mcp` in any tool that supports the Model Context Protocol.
