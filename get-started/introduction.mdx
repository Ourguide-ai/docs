## Installation

Add the Ourguide widget to your site with a single script tag:

<Tabs>
  <Tab title="Script Tag">
    ```html
    <script
      src="https://dashboard.ourguide.ai/ourguide-b2b-widget.iife.js"
      data-api-url="https://dashboard.ourguide.ai"
      data-product-id="YOUR_PRODUCT_ID"
    ></script>
    ```
  </Tab>

  <Tab title="React SDK">
    ```bash
    npm install @ourguide-ai/client @ourguide-ai/ui
    ```

    ```tsx
    import { OurguideWidget } from '@ourguide-ai/ui';
    import '@ourguide-ai/ui/styles.css';

    <OurguideWidget productId="YOUR_PRODUCT_ID" apiUrl="https://dashboard.ourguide.ai" />
    ```
  </Tab>
</Tabs>

## Core Features

| Feature                 | What it does                                                          |
| ----------------------- | --------------------------------------------------------------------- |
| **Knowledge Base**      | Agent learns from your docs, website                    |
| **API Integration**     | Agent calls your API to take actions (check orders, cancel subscriptions) |
| **Identity Verification** | Agent knows who the user is and can act on their behalf             |
