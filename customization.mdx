Make the Ourguide widget match your brand and product experience.

## Widget Appearance

Customize the widget via data attributes on the script tag:

```html
<script
  src="https://api.ourguide.ai/static/ourguide-widget.js"
  data-api-url="https://api.ourguide.ai"
  data-product-id="YOUR_PRODUCT_ID"
  data-primary-color="#16A34A"
  data-position="bottom-right"
  data-greeting="Hi! How can I help you?"
></script>
```

## Available Options

| Attribute | Description | Default |
|---|---|---|
| `data-primary-color` | Widget accent color | `#16A34A` |
| `data-position` | Widget position (`bottom-right`, `bottom-left`) | `bottom-right` |
| `data-greeting` | Initial greeting message | `"How can I help you?"` |
| `data-avatar` | URL to agent avatar image | Ourguide default |
| `data-z-index` | CSS z-index for the widget | `9999` |

## Agent Instructions

In your dashboard under **Settings > Agent**, you can configure:

- **System prompt** — Define the agent's personality and behavior
- **Greeting message** — The first message users see
- **Fallback message** — What to say when the agent doesn't know the answer
- **Handoff trigger** — When to escalate to a human
