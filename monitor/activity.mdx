View all conversations your agent has with users at [dashboard.ourguide.ai/conversations](https://dashboard.ourguide.ai/conversations).

## Layout

- **Left panel** — Conversation list with filters
- **Right panel** — Selected conversation details

## Filters

| Filter            | Options              |
| ----------------- | -------------------- |
| **Date range**    | Start/end date       |
| **Status**        | Success, Error       |
| **Message count** | Number of messages   |
| **Search**        | Text search          |

## Conversation Details

### Chat Tab

Full conversation as users see it:

- User messages
- Agent responses
- Tool calls and results
- Timestamps

### Tool Calls

Tool calls appear inline with parameters and results:

```
🔧 Calling: getOrderStatus
   Parameters: {"orderId": "ORD-12345"}
✅ Result: {"status": "shipped"}
```

### Details Tab

| Field               | Description        |
| ------------------- | ------------------ |
| **Conversation ID** | Unique identifier  |
| **Duration**        | Start to end time  |
| **Messages**        | Total count        |
| **Tokens**          | Usage              |
| **Cost**            | Estimated (if enabled) |

### Traces Tab

Raw debugging data — useful for complex issues.

## Use Activity To

- **Identify gaps** — Questions agent can't answer → add to knowledge base
- **Spot issues** — Errors, tool failures
- **Track volume** — Conversation trends over time

## Export

Click the download button to export conversation data.
