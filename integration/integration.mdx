Integrations let your agent take actions — not just answer questions. Connect your REST API so the agent can check orders, create tickets, process refunds, and more.

## How It Works

1. Upload your OpenAPI/Swagger spec
2. Configure authentication (API key, Bearer token, or JWT Forward)
3. Enable the endpoints you want the agent to use
4. The agent automatically calls the right endpoint based on the user's question

## Example Actions

- Check order status
- Create support tickets
- Look up customer info
- Process refunds
- Update account settings
- Cancel subscriptions
